# 小项目集合

预览地址：[https://moohng.com/demo/](https://moohng.com/demo/)

## 本地运行

```bash
$ npm run dev **/*
```

## 发布

1. 子目录创建 `vite.config.js` 配置文件，修改打包输出目录到根目录的 `docs/**` 下；
2. 运行 `build` 命令；

```bash
$ npm run build **/**
```
